doubanFM
========

doubanFM desktop client with Qt
###全局快捷键 global shortcut
Alt + p 暂停/播放 Pause/Play

Alt + s 下一首	NextSong

Alt + t 不再播放 Never Play

Alt + r 喜欢/不喜欢这首歌 Rate/Unrate this song
